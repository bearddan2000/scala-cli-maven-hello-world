# scala-cli-maven-hello-world

## Description
A POC for scala maven build.

## Tech stack
- maven
- scala

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
